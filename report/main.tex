\section{Giới thiệu}
Đây là nội dung mẫu để kiểm tra xem môi trường LaTeX của bạn đã hoạt động ổn định chưa. 
Nếu bạn nhìn thấy file PDF bên phải (hoặc khi mở ra), chúc mừng bạn! \textbf{Cài đặt đã thành công}.

\section{Thử nghiệm công thức toán}
LaTeX rất mạnh về hiển thị công thức toán học. Ví dụ phương trình Euler nổi tiếng:
\begin{equation}
    e^{i\pi} + 1 = 0
\end{equation}

Hoặc tích phân:
\[
    \int_{-\infty}^{\infty} e^{-x^2} \,dx = \sqrt{\pi}
\]

\section{Thử nghiệm danh sách}
Một vài việc cần làm tiếp theo:
\begin{enumerate}
    \item Lưu file này (`Ctrl + S`).
    \item Chờ extension tự động build.
    \item Mở file PDF (`Ctrl + Alt + V`).
    \item Thử sửa nội dung và xem PDF cập nhật.
\end{enumerate}
